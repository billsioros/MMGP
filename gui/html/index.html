Multiagent Multitarget Geospatial Pathfinding

  • Firstly, information, crucial in determining the best route for every bus, is being retrieved from the school's database.

  • For each of the students, given that the information retrieved from the database was valid, we use a Geospatial API in order to determine where we should pick them up from and how time consuming is going to be.

  • The students are being clustered into groups based on how close they are to each other as well as to the school and if they fall into the same time-window.

  • Lastly, the best path for the bus to follow is determined among each group of students
